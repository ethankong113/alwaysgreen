About this site

Stock Analyzer is a web application that allows you to analyze selected public stocks. It uses Node.JS as the API to collect and server financial data. On the frontend, it uses jQuery for DOM manipulation and D3.js for data visualization. Price history data is provided by Quandl's API and fundamentals data is provided by Google Finance.
